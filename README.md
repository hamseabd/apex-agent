# Apex

> Self-hosted Telegram health accountability bot.  
> Protocol-driven. Dynamic tools. AWS Lambda + Bedrock.

Send `/setup` to configure your personal health protocol. The bot generates tools for whatever you track.

## Quick start

```bash
# Clone, configure, deploy
git clone https://github.com/your-username/apex.git
cd apex
cp apex.example.yaml apex.yaml   # edit with your protocol
./scripts/bootstrap.sh
```
